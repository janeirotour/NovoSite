Create a global Hotel Accommodation Add-On system for all travel packages on the website.

Do not redesign the website.
Do not change the current structure, branding, colors or existing package layouts.
Only add hotel accommodation management and package integration.

MAIN GOAL

Allow customers to add hotel accommodation to any current or future travel package.

Create one centralized hotel catalog in the Admin Dashboard.

All active hotels in this catalog must automatically become available as optional accommodation choices across every package on the website, without requiring manual configuration on each package page.

CURRENCY

All hotel prices listed below are in Brazilian Real:

BRL / R$

ROOM BASIS

All listed prices are:

* Per room
* Per night
* Based on double occupancy
* Queen-bed room
* Subject to availability and confirmation

The hotel price must not be calculated per person unless manually configured otherwise by the administrator.

PRICE CALCULATION

For regular dates:

Hotel total =
Nightly room price × Number of rooms × Number of nights

Do not multiply the room rate by the number of guests when occupancy is within the configured room limit.

Show separately:

* Package price
* Hotel price
* Optional extras
* Final estimated total

SPECIAL DATES

Create a Special Season pricing system.

Special seasons should include:

* Carnival
* New Year’s Eve
* National holidays
* Major local events
* Custom date ranges defined by the admin

When selected dates fall within a special season:

* Do not guarantee the displayed reference price.
* Clearly show:
  "Special-season rate — final price and availability require confirmation."
* Allow the customer to submit the package as a booking request or quote request.
* Do not finalize an automatic confirmed hotel reservation unless availability and price have been confirmed.

HOTEL AVAILABILITY AND SUBSTITUTION

Display this notice during selection and in the booking summary:

"If the selected hotel is unavailable, Janeiro Tour & Travel may offer an alternative hotel of the same or a higher category in a comparable location. Any substitution and price difference will be presented to the customer for approval before confirmation."

Do not replace the hotel automatically without customer approval.

The admin must be able to:

* Mark a hotel as available
* Mark a hotel as unavailable
* Mark a hotel as available on request
* Suggest an equivalent hotel
* Add a replacement hotel
* Record any price difference
* Request customer approval before confirming the substitution
