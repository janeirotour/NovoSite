Update the Admin Dashboard to make the entire website fully editable without coding.

Do not redesign the website.
Do not change the existing visual identity.
Only improve the admin system.

Create a simple visual page editor similar to Elementor, but do not build an overly complicated custom page builder from scratch.

The editor should be clean, modern, user-friendly and easy for a non-technical person to use.

Visual Editor Requirements:

* Allow me to edit page content without coding
* Allow me to click on sections and edit text directly
* Allow me to update headings, paragraphs, buttons and CTAs
* Allow me to upload images directly from my computer
* Allow me to replace hero images, destination images, tour images, experience images and blog images
* Allow me to edit sections visually
* Allow me to edit menu labels
* Allow me to edit footer content
* Allow me to preview changes before publishing
* Add Save, Preview and Publish buttons

The visual editor must work for absolutely all pages and content types, including:

* Homepage
