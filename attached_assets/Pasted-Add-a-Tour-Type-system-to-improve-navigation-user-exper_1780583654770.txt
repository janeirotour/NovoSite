Add a Tour Type system to improve navigation, user experience and conversions.

Do not redesign the website.
Do not change the existing branding, colors or structure.

Create a clear distinction between Shared Tours and Private Tours throughout the website.

* All Experiences
* Private Tours
* Shared Tours
* Airport Transfers

* Bahia
* Foz do Iguaçu
* Recife

* Cultural Experiences

* Food Experiences

Examples:

Shared Tour
Private Tour

* Most Popular

Add a new field for each experience:

Tour Type

* Shared Tour
* Private Tour

Premium Badge

* None
* Most Personalized Experience
* Best for Families

* Best Value
* Small Groups
